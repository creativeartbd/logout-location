Choose where to redirect after logout from your WordPress dashboard.

== Description ==

#General
=======
In this general tab, you can find an option to choose a role type. There are 2 roles are available which are:

1. Any Roles
2. Multiple Roles

Any Roles
=========
If you choose Any Roles then it will show an option that where you want to redirect all of your WordPress roles.

If you choose Multiple Roles then it will show a list of your WordPress roles along with different options. Here you can individually choose which role will redirect to which location.

Where to redirect
=================
Currently, there are 7 options that you can choose to redirect. Which are below:

1. Post
2. Page
3. Custom Post
4. Category
5. Tag
6. Custom Link
7. Product (Woocommerce)

Page
====
If you choose a page it will show all available pages from your WordPress site.

Custom Link
===========
If you a choose custom link then it will show you an input box where you need to provide the custom link.

Post
=====
If you choose a post all of your posts will be shown there. Please note that this post will only show WordPress default posts.

Custom Post
===========
If you choose a custom post then you can choose a list of all custom posts registered on your WordPress website.

Product (WooCommerce)
=====================
If you choose Product (WooCommerce) and if the WooCommerce plugin is installed then this option will show you all the product list.

User page
=========
If you choose User then all of your site users will be shown there.

Category
========
If you choose a category then all of your including custom post categories will be shown there.

Tag
====
If you choose tag then all of your registered tags will be shown on this list.

#Access List
===========
This plugin has an option to choose who can use this plugin by going to the Access List tab. This Access List tab will show all of your WordPress roles. You can choose any one of them or multiple ones.

#Logout History
===============
Every user logout from your WordPress website will be stored. You can view all of your logout histories from the Logout History tab. Anytime you can clear all the history.
